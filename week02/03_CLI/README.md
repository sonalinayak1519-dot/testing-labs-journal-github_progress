# 🐧 Raspberry Pi CLI Activities for Beginners

These activities help you practise the essential command-line skills for navigating folders, managing files, and checking basic networking information on a Raspberry Pi.
These activities help you practise the essential command-line skills needed for working effectively on a Raspberry Pi. For navigating folders, you will use commands such as `pwd` to print the current working directory, `cd` to change into a specified folder, and `ls` to list all files and folders in the current directory.

For managing files, you will use commands such as `mkdir` to create a new folder, `touch` to create a new empty file, `cp` to copy a file, `mv` to move or rename a file, and `rm` to delete a file. These commands allow you to organise and manage your files directly from the terminal without needing a graphical interface.

For checking basic networking information, commands such as `ifconfig` or `ip a` display network interface details including the IP address, `ping` tests connectivity to another device or website, and `hostname -I` shows the IP address of the Raspberry Pi. Together, these skills form the foundation for more advanced tasks such as setting up servers, automating processes, and managing the system remotely.
