| Component | Function | Key Specification |
|----------|----------|------------------|
| CPU | The CPU (Central Processing Unit) is the brain of the computer, responsible for processing instructions and performing calculations. It controls the flow of data between all other components and is what allows software and programs to run.| Key Specifications: Clock speed (GHz), number of cores/threads, cache size, and TDP (thermal design power) are the main specs that determine a CPU's performance and power consumption.|
| RAM |RAM (Random Access Memory) is the computer's short-term memory, temporarily storing data that the CPU is actively using so it can be accessed quickly. It allows multiple programs to run at the same time, and the more RAM a computer has, the smoother and faster it can perform. | Key Specifications: Capacity (GB), speed (MHz), type (DDR4/DDR5), and number of channels are the main specs that determine RAM's performance and compatibility with a system.|
| Storage | Storage is where the computer permanently saves all data, including the operating system, software, and personal files, even when the power is turned off. It comes in two main types — HDD (Hard Disk Drive) and SSD (Solid State Drive), with SSDs being faster and more commonly used in modern systems.|Key Specifications: Capacity (GB/TB), read/write speed (MB/s), interface type (SATA/NVMe), and form factor (2.5"/M.2) are the main specs that determine a storage drive's performance and compatibility.|
| Motherboard |The motherboard is the main circuit board of the computer that connects and allows communication between all components such as the CPU, RAM, storage, and graphics card. It acts as the backbone of the system, distributing power and data to every part of the computer. |Key Specifications: Form factor (ATX/Micro-ATX), socket type (determines CPU compatibility), chipset, number of RAM slots, and available expansion slots (PCIe) are the main specs that determine a motherboard's compatibility and capability.|
| Power Supply |The power supply unit (PSU) converts AC power from the wall outlet into DC power that the computer's components can use, providing the correct voltage to each part of the system. Without it, none of the components would receive the electricity they need to function. |Key Specifications: Wattage (W), efficiency rating (80+ Bronze/Gold/Platinum), modular/non-modular design, and output voltages (12V/5V/3.3V) are the main specs that determine a power supply's performance and compatibility with a system. |
| Cooling |Cooling keeps the computer's components, particularly the CPU and GPU, from overheating during operation by dissipating the heat they generate. It comes in two main types — air cooling (using heatsinks and fans) and liquid cooling (using water blocks and radiators), with liquid cooling being more effective for high-performance systems. | Key Specifications: Cooling type (air/liquid), fan size (mm), TDP support (W), noise level (dBA), and radiator size (mm) are the main specs that determine a cooling system's performance and compatibility with a system.|
