# Part 1 – Hardware vs Software

## Explain Hardware
Write **at least 50 words** explaining what hardware is.
---Hardware refers to the physical components of a computer system that can be seen and touched. It includes devices such as the processor (CPU), memory (RAM), hard drive or SSD, motherboard, keyboard, mouse, monitor, and printer. Hardware works together with software to perform tasks, process data, and run applications. Without hardware, a computer would not be able to function.

## Explain Software
Write **at least 50 words** explaining what software is.
---Software is the collection of programs, instructions, and data that tell a computer how to perform specific tasks. Unlike hardware, software cannot be physically touched. Examples include operating systems such as Windows and macOS, as well as applications like word processors, web browsers, and games. Software works with hardware to enable a computer to function and perform tasks for users.

## How Do Hardware and Software Interact?
Write **at least 50 words** describing how they interact.
Hardware and software interact by working together to perform tasks on a computer. Software provides the instructions, while hardware carries out those instructions. For example, when you open a word-processing program, the software tells the computer what to do, and the hardware components such as the CPU, RAM, and storage process the data and display the program on the monitor. Without software, hardware would have no instructions to follow, and without hardware, software would have no device on which to run. Together, they enable the computer to function effectively.
